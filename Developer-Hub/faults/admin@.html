Tutorials

HTML Tutorial

HTML Syntax

HTML (HyperText Markup Language) is the standard markup language for documents designed to be displayed in a web browser.

Basic HTML Document Structure

<!DOCTYPE html>
    <html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        <h1>My First Heading</h1>
        <p>My first paragraph.</p>
    </body>
    </html>

HTML Elements & Attributes

HTML elements are the building blocks of HTML pages. Elements can have attributes that provide additional information.

Element with Attributes

<a href="https://www.example.com" target="_blank">
        Visit Example.com
    </a>

HTML Headings & Paragraphs

Headings are defined with the <h1> to <h6> tags. Paragraphs are defined with the <p> tag.

HTML Styles & Formatting

HTML provides several elements for text formatting: <b>, <i>, <strong>, <em>, <mark>, etc.

HTML Comments

Comments are not displayed in the browser, but they can help document your HTML source code.

<!-- This is a comment -->
    <p>This is a paragraph.</p>
    <!-- Remember to add more content here -->

HTML Colors

Colors can be specified using color names, HEX values, RGB, RGBA, HSL, or HSLA values.

HTML Links

Links are defined with the <a> tag. The href attribute specifies the URL of the page the link goes to.

HTML Tables

HTML tables are defined with the <table> tag. A table is divided into rows (<tr>), and each row is divided into data cells (<td>).

HTML Lists

HTML lists can be unordered (<ul>), ordered (<ol>), or description lists (<dl>).

HTML Classes & ID

The class attribute can be used on any HTML element. The id attribute specifies a unique id for an element.

HTML Iframes

An iframe is used to display a web page within a web page.

HTML File Paths

File paths describe the location of a file in a website's folder structure.

HTML Head Element

The <head> element contains meta information about the HTML document.

HTML Layout Elements

HTML5 introduced semantic elements that define different parts of a web page.

HTML Responsive Design

Responsive web design makes your web page look good on all devices.

HTML Charset & URL Encode

The charset attribute specifies the character encoding for the HTML document.

CSS Tutorial

CSS Syntax

A CSS rule consists of a selector and a declaration block.

CSS Syntax Example

p {
        color: red;
        text-align: center;
    }

CSS Selectors

CSS selectors are used to "find" (or select) HTML elements you want to style.

CSS Colors

Colors in CSS can be specified by color names, HEX values, RGB, RGBA, HSL, or HSLA values.

CSS Backgrounds

CSS background properties are used to define background effects for elements.

CSS Borders, Margins & Padding

The CSS box model describes the rectangular boxes generated for elements.

CSS Height/Width & Box Model

The height and width properties are used to set the height and width of an element.

CSS Text & Fonts

CSS has many properties for formatting text and fonts.

CSS Icons, Links & Lists

CSS can style icons, links, and lists to create beautiful navigation and content.

CSS Tables & Display

The display property specifies the display behavior of an element.

CSS Position & Overflow

The position property specifies the type of positioning method used for an element.

CSS Float & Align

The float property is used for positioning and formatting content.

CSS Pseudo-classes & Pseudo-elements

Pseudo-classes are used to define a special state of an element.

CSS Opacity & Navigation Bar

The opacity property specifies the opacity/transparency of an element.

CSS Image Gallery & Forms

CSS can create beautiful image galleries and styled forms.

CSS Animations & Website Layouts

CSS animations allow animation of most HTML elements without JavaScript.

CSS Units

CSS has several different units for expressing length, such as px, em, rem, %, vw, vh, etc.

JavaScript Tutorial

JavaScript Syntax

JavaScript syntax is the set of rules that define a correctly structured JavaScript program.

Basic JavaScript Syntax

// This is a comment
    let message = "Hello, World!";
    console.log(message);

JavaScript Comments

JavaScript comments can be used to explain JavaScript code and make it more readable.

JavaScript Variables

Variables are containers for storing data values. JavaScript uses var, let, and const to declare variables.

JavaScript Operators

Operators are used to assign values, compare values, perform arithmetic operations, and more.

JavaScript Data Types

JavaScript has 8 data types: String, Number, BigInt, Boolean, Undefined, Null, Symbol, and Object.

JavaScript Functions

A JavaScript function is a block of code designed to perform a particular task.

JavaScript Events

HTML events are "things" that happen to HTML elements, like when a user clicks a button.

JavaScript Objects

Objects are variables too, but they can contain many values written as name:value pairs.

JavaScript Strings & Numbers

JavaScript strings are for storing and manipulating text. JavaScript has only one type of number.

JavaScript Methods

Methods are actions that can be performed on objects, like string methods or array methods.

JavaScript Arrays

JavaScript arrays are used to store multiple values in a single variable.

JavaScript Arithmetic & Math

JavaScript arithmetic operators perform arithmetic on numbers. The Math object provides mathematical functions.

JavaScript Random & Booleans

The Math.random() method returns a random number between 0 and 1. Booleans can only have two values: true or false.

JavaScript Comparisons

Comparison operators are used in logical statements to determine equality or difference between variables or values.

JavaScript Loops: For & While

Loops can execute a block of code a number of times, or while a condition is true.

JavaScript Conditions

Conditional statements are used to perform different actions based on different conditions.

HTML Reference

HTML Tag List

Tag Description
<html> Defines the root of an HTML document
<head> Contains metadata/information for the document
<title> Defines a title for the document
<body> Defines the document's body
<h1> to <h6> Defines HTML headings
<p> Defines a paragraph
<a> Defines a hyperlink
<img> Defines an image
<ul>, <ol>, <li> Defines unordered/ordered lists and list items
<table>, <tr>, <td> Defines a table, table row, and table data

HTML Attributes

Common HTML attributes include: id, class, style, src, href, alt, title, width, height, etc.

HTML Events

Common events: onclick, onmouseover, onload, onchange, onsubmit, etc.

HTML Colors

Color names, HEX values, RGB, RGBA, HSL, HSLA values.

HTML Audio/Video

<audio> and <video> tags for embedding media content.

HTML Canvas & Doctypes

<canvas> tag for drawing graphics. <!DOCTYPE> declaration defines document type.

URL Encoding

URL encoding converts characters into a format that can be transmitted over the Internet.

CSS Reference

CSS Selectors

Selector Example Description
Element selector p Selects all <p> elements
ID selector #header Selects the element with id="header"
Class selector .intro Selects all elements with class="intro"
Universal selector * Selects all elements
Attribute selector a[target] Selects all <a> elements with a target attribute

CSS Functions

Common functions: rgb(), rgba(), hsl(), hsla(), calc(), var(), url(), linear-gradient(), etc.

CSS Units

Absolute: px, pt, cm, mm, in. Relative: %, em, rem, vw, vh, vmin, vmax.

CSS Colors

Color names, HEX, RGB, RGBA, HSL, HSLA, currentColor, transparent.

Web Safe Fonts

Arial, Helvetica, Times New Roman, Times, Courier New, Courier, Verdana, Georgia, etc.

JavaScript Reference

JavaScript Objects

Object Description
String Represents and manipulates a sequence of characters
Number Represents numeric values
Array Represents a collection of elements
Date Represents dates and times
Math Provides mathematical functions and constants
JSON Parses and stringifies JSON data
Promise Represents the eventual completion of an async operation

HTML DOM Objects

Document, Element, Event, NodeList, HTMLCollection, etc.
