Introduction to JavaScript

JavaScript is a versatile programming language that enables interactive web pages. It's an essential part of web development alongside HTML and CSS.

What is JavaScript?

  • Client-side scripting language: Runs in the user's web browser
  • Dynamic and interactive: Makes web pages responsive to user actions
  • Object-oriented: Supports object-oriented programming paradigms
  • Multi-paradigm: Supports procedural, functional, and event-driven programming
  • Standardized as ECMAScript: ES6 (2015) introduced major improvements

  1. Include JavaScript in HTML using <script> tags

Variables & Data Types

Variables are containers for storing data values. JavaScript has dynamic typing, meaning variables can hold different types of data.

Variable Declaration

// Three ways to declare variables
var oldVariable = "I'm the old way"; // Function scope
let modernVariable = "I'm block-scoped"; // Preferred for variables that change
const constantVariable = "I cannot be reassigned"; // Preferred for constants

// Variable naming conventions
let camelCaseVariable = "JavaScript uses camelCase";
const UPPER_SNAKE_CASE = "For constants";
let $specialVariable = "$ and _ are allowed";

Operators & Expressions

Operators perform operations on variables and values. Expressions are combinations of values, variables, and operators that evaluate to a value.

Arithmetic Operators

Perform mathematical operations:

let sum = 10 + 5;    // 15 (Addition)
let difference = 10 - 5; // 5 (Subtraction)
let product = 10 * 5;  // 50 (Multiplication)
let quotient = 10 / 5;  // 2 (Division)
let remainder = 10 % 3;  // 1 (Modulus)
let exponent = 2 ** 3;  // 8 (Exponentiation)

Comparison Operators

Compare values and return true or false:

5 == "5"      // true (loose equality)
5 === "5"     // false (strict equality)
5 != "5"      // false (loose inequality)
5 !== "5"     // true (strict inequality)
10 > 5       // true (greater than)
10 <= 10      // true (less than or equal)

Logical Operators

Combine or invert boolean values:

let andResult = true && false;  // false (AND)
let orResult = true || false;   // true (OR)
let notResult = !true;           // false (NOT)

// Short-circuit evaluation
let value = null || "default"; // "default"

Assignment Operators

JavaScript provides shorthand assignment operators that combine arithmetic operations with assignment:

let x = 10;
x += 5;   // x = x + 5 (now 15)
x -= 3;   // x = x - 3 (now 12)
x *= 2;   // x = x * 2 (now 24)
x /= 4;   // x = x / 4 (now 6)
x %= 4;   // x = x % 4 (now 2)

Control Flow

Control flow statements determine the order in which code is executed. They allow your program to make decisions and repeat actions.

Conditional Statements

// if statement
let age = 18;

if (age >= 18) {
  console.log("You are an adult");
}

// if...else statement
if (age >= 18) {
  console.log("You can vote");
} else {
  console.log("You cannot vote yet");
}

// if...else if...else statement
let score = 85;

if (score >= 90) {
  console.log("Grade: A");
} else if (score >= 80) {
  console.log("Grade: B");
} else if (score >= 70) {
  console.log("Grade: C");
} else {
  console.log("Grade: F");
}

// switch statement
let day = "Monday";

switch (day) {
  case "Monday":
    console.log("Start of the work week");
    break;
  case "Friday":
    console.log("Almost weekend!");
    break;
  default:
    console.log("It's a regular day");
}

Loops

// for loop
for (let i = 0; i < 5; i++) {
  console.log("Iteration:", i);
}

// while loop
let count = 0;
while (count < 3) {
  console.log("Count:", count);
  count++;
}

// do...while loop (executes at least once)
let num = 5;
do {
  console.log("Number:", num);
  num--;
} while (num > 0);

// for...of loop (for arrays)
let colors = ["red", "green", "blue"];
for (let color of colors) {
  console.log(color);
}

// for...in loop (for object properties)
let person = { name: "John", age: 30 };
for (let key in person) {
  console.log(key, ":", person[key]);
}

Functions

Functions are reusable blocks of code that perform a specific task. They help organize code, avoid repetition, and make programs more modular.

Function Declaration & Expression

// Function declaration (hoisted)
function greet(name) {
  return "Hello, " + name + "!";
}

// Function expression
const add = function(a, b) {
  return a + b;
};

// Arrow function (ES6+)
const multiply = (a, b) => a * b;

// Arrow function with multiple statements
const getGrade = (score) => {
  if (score >= 90) return "A";
  if (score >= 80) return "B";
  return "C";
};

// Calling functions
console.log(greet("Alice"));      // "Hello, Alice!"
console.log(add(5, 3));          // 8
console.log(multiply(4, 5));    // 20
console.log(getGrade(85));       // "B"

Function Parameters & Default Values

// Default parameters (ES6+)
function createUser(name, age = 18, isAdmin = false) {
  return {
    name: name,
    age: age,
    isAdmin: isAdmin
  };
}

// Rest parameters (...)
function sum(...numbers) {
  let total = 0;
  for (let num of numbers) {
    total += num;
  }
  return total;
}

// Function calling examples
const user1 = createUser("John");           // age=18, isAdmin=false
const user2 = createUser("Jane", 25);       // isAdmin=false
const user3 = createUser("Admin", 30, true);

console.log(sum(1, 2, 3));          // 6
console.log(sum(10, 20, 30, 40));  // 100

// Higher-order function example
function applyOperation(a, b, operation) {
  return operation(a, b);
}

const result = applyOperation(5, 3, (x, y) => x * y);
console.log(result);  // 15

Arrays & Objects

Arrays and objects are two fundamental data structures in JavaScript for organizing and storing collections of data.

Arrays

// Array creation
let fruits = ["apple", "banana", "orange"];
let numbers = [1, 2, 3, 4, 5];
let mixed = ["text", 42, true, null];

// Accessing elements
console.log(fruits[0]);   // "apple" (first element)
console.log(fruits[fruits.length - 1]);  // "orange" (last element)

// Array methods
fruits.push("grape");      // Add to end
fruits.pop();             // Remove from end
fruits.unshift("kiwi");   // Add to beginning
fruits.shift();           // Remove from beginning

// Common array operations
let squared = numbers.map(num => num * num);
let evens = numbers.filter(num => num % 2 === 0);
let sum = numbers.reduce((total, num) => total + num, 0);

console.log(squared);  // [1, 4, 9, 16, 25]
console.log(evens);    // [2, 4]
console.log(sum);      // 15

Objects

// Object creation
let person = {
  firstName: "John",
  lastName: "Doe",
  age: 30,
  isStudent: false,
  hobbies: ["reading", "gaming"],
  address: {
    street: "123 Main St",
    city: "Anytown"
  },
  greet: function() {
    return "Hello, my name is " + this.firstName;
  }
};

// Accessing properties
console.log(person.firstName);      // Dot notation
console.log(person["lastName"]);     // Bracket notation
console.log(person.address.city);  // Nested objects
console.log(person.greet());         // Method call

// Adding and modifying properties
person.email = "john@example.com";   // Add new property
person.age = 31;                    // Modify existing property
delete person.isStudent;          // Delete property

// Object methods
let keys = Object.keys(person);      // Get all keys
let values = Object.values(person);  // Get all values
let entries = Object.entries(person); // Get key-value pairs

// Object destructuring (ES6+)
const { firstName, age } = person;
console.log(firstName, age);  // "John" 31

Create a simple program that manages a list of books with the following functionality:

Requirements

  • Write a function to add a new book to the array
  • Write a function to find books by a specific author
  • Write a function to get all books published after a given year
  • Write a function to display all books in a formatted way
Show Solution

Book Management Solution

// Book management program
let books = [
  { title: "The Great Gatsby", author: "F. Scott Fitzgerald", year: 1925 },
  { title: "To Kill a Mockingbird", author: "Harper Lee", year: 1960 },
  { title: "1984", author: "George Orwell", year: 1949 }
];

// Add a new book
function addBook(title, author, year) {
  books.push({ title, author, year });
}

// Find books by author
function findBooksByAuthor(authorName) {
  return books.filter(book => book.author === authorName);
}

// Get books published after a year
function getBooksAfterYear(year) {
  return books.filter(book => book.year > year);
}

// Display all books
function displayBooks() {
  books.forEach((book, index) => {
    console.log(`${index + 1}. "${book.title}" by ${book.author} (${book.year})`);
  });
}

// Test the functions
addBook("Brave New World", "Aldous Huxley", 1932);
const orwellBooks = findBooksByAuthor("George Orwell");
const modernBooks = getBooksAfterYear(1950);
displayBooks();
