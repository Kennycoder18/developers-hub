CSS Basic Syntax

CSS (Cascading Style Sheets) uses a simple syntax to style HTML elements. A CSS rule consists of a selector and a declaration block.

CSS Rule Structure

selector {
property : value ;
}
Basic CSS Syntax Example 
h1 {
  color: #2dd4bf;
  font-size: 2.5rem;
  text-align: center;
}

/* This is a CSS comment */
.container {
  width: 80%;
  margin: 0 auto;
  padding: 20px;
}

#main-header {
  background-color: #0d9488;
  color: white;
  padding: 1rem;
}
Key Points:
  • Selectors target HTML elements to style
  • Properties define what aspect to change (color, size, etc.)
  • Values specify how to change the property
  • Each declaration ends with a semicolon
  • Multiple declarations go inside curly braces

CSS Selectors

Selectors are patterns used to select the elements you want to style. CSS offers a variety of selector types for different needs.

Selector Type Example Description
Element Selector p { } Selects all <p> elements
Class Selector .button { } Selects elements with class="button"
ID Selector #header { } Selects element with id="header"
Universal Selector * { } Selects all elements
Attribute Selector [type="text"] { } Selects elements with specific attribute
Pseudo-class a:hover { } Selects elements in special state
Descendant Selector div p { } Selects <p> inside <div>
Selector Examples 
/* Element selector */
h1 {
  color: #0d9488;
}

/* Class selector */
.highlight {
  background-color: #f0fdfa;
}

/* ID selector */
#main-content {
  width: 100%;
}

/* Multiple elements */
h1, h2, h3 {
  font-family: 'Arial', sans-serif;
}

/* Descendant selector */
nav ul li {
  display: inline-block;
}

/* Pseudo-class */
a:hover {
  color: #10b981;
}

Common CSS Properties

CSS properties control the visual presentation of HTML elements. Here are the most commonly used properties:

Color & Background

  • color - Text color
  • background-color
  • background-image
  • opacity - Transparency

Typography

  • font-family
  • font-size
  • font-weight - Boldness
  • text-align

Box Model

  • width & height
  • margin - Outer spacing
  • padding - Inner spacing
  • border

Layout

  • display - Block, inline, flex
  • position
  • flex-direction
  • grid-template-columns
Property Examples 
/* Color and typography */
.card {
  color: #1f2937;
  background-color: #f9fafb;
  font-family: 'Segoe UI', Tahoma, sans-serif;
  font-size: 1rem;
}

/* Box model properties */
.container {
  width: 90%;
  max-width: 1200px;
  margin: 0 auto;
  padding: 2rem;
  border: 2px solid #0d9488;
  border-radius: 10px;
}

/* Flexbox layout */
.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  gap: 1.5rem;
}

Styling HTML Elements

CSS can style any HTML element. Here's how to style common elements with practical examples:

Element Styling Examples

button.css 
.styled-button {
  background: linear-gradient(135deg, #0d9488, #10b981);
  color: white;
  border: none;
  padding: 12px 24px;
  border-radius: 8px;
  font-size: 1rem;
  font-weight: 600;
  cursor: pointer;
  transition: all 0.3s ease;
}

.styled-button:hover {
  transform: translateY(-2px);
  box-shadow: 0 4px 12px rgba(13, 148, 136, 0.3);
}

CSS Units & Values

CSS supports various units for measurements and different value types for properties.

Unit Type Example Description
Absolute Units px, cm, in Fixed size regardless of screen
Relative Units %, em, rem Relative to parent or root element
Viewport Units vw, vh, vmin Relative to viewport size
Color Values #0d9488, rgb(), hsl() Different color representation formats
Function Values calc(), linear-gradient() CSS functions that return values
Unit Examples 
/* Absolute units */
.fixed-size {
  width: 300px;
  padding: 20px;
}

/* Relative units */
.responsive {
  width: 80%;
  font-size: 1.2rem;
  margin: 2em;
}

/* Viewport units */
.full-screen {
  width: 100vw;
  height: 100vh;
}

/* Color values */
.color-examples {
  color: #0d9488; /* Hex */
  background-color: rgb(14, 165, 233); /* RGB */
  border-color: hsl(162, 73%, 46%); /* HSL */
}

/* CSS functions */
.calculated {
  width: calc(100% - 40px);
  background: linear-gradient(135deg, #0d9488, #10b981);
}

Key Takeaways

You've learned the fundamentals of CSS syntax and elements! Here's what you should remember:
  • CSS rules consist of selectors and declaration blocks
  • Selectors target HTML elements (by element, class, ID, etc.)
  • Properties define what to style, values define how
  • CSS can style any HTML element with various properties
  • Use appropriate units and values for responsive design

Continue practicing by:

  • Creating your own CSS styles for HTML elements
  • Experimenting with different selectors and properties
  • Building responsive layouts with Flexbox and Grid
  • Studying CSS frameworks like Bootstrap or Tailwind
