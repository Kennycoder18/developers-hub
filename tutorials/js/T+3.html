Objects vs Arrays: Understanding the Difference

Objects and arrays are both used to store collections of data, but they serve different purposes and have different characteristics.

Objects: Key-Value Collections

  • Structure: Unordered collection of key-value pairs
  • Access: By property name (key)
  • Use when: You need named properties, represent entities
  • Example: User profile, configuration settings
  • Type: Reference type (stores memory reference)
// Object example
const user = {
  name: "John Doe",
  age: 30,
  email: "john@example.com",
  isAdmin: false
};

Arrays: Ordered Lists

  • Structure: Ordered list of values with numeric indices
  • Access: By index (position)
  • Use when: You need ordered data, lists of items
  • Example: Shopping list, scores, todos
  • Type: Reference type (special kind of object)
// Array example
const fruits = [
  "apple",
  "banana",
  "orange",
  "grape"
];

Creating Objects & Arrays

There are multiple ways to create objects and arrays in JavaScript, each with its own use cases.

Object Creation Methods

// 1. Object literal (most common)
const person = {
  name: "Alice",
  age: 25,
  greet: function() {
    return "Hello!";
  }
};

// 2. Object constructor
const car = new Object();
car.make = "Toyota";
car.model = "Camry";
car.year = 2020;

// 3. Object.create() - for inheritance
const animal = {
  type: "mammal",
  sound: "...",
  makeSound: function() {
    console.log(this.sound);
  }
};

const dog = Object.create(animal);
dog.sound = "Woof!";

// 4. Constructor function
function Book(title, author, year) {
  this.title = title;
  this.author = author;
  this.year = year;
}

const book1 = new Book("1984", "George Orwell", 1949);

// 5. ES6 Class
class Product {
  constructor(name, price) {
    this.name = name;
    this.price = price;
  }
  
  getDescription() {
    return `${this.name} - $${this.price}`;
  }
}

const product1 = new Product("Laptop", 999);

Array Creation Methods

// 1. Array literal (most common)
const colors = ["red", "green", "blue"];

// 2. Array constructor
const numbers = new Array(1, 2, 3, 4, 5);

// 3. Array with length (creates empty slots)
const emptyArray = new Array(5); // Creates array with 5 empty slots

// 4. Array.from() - from array-like or iterable objects
const arrayFromString = Array.from("hello"); // ["h", "e", "l", "l", "o"]
const arrayFromSet = Array.from(new Set([1, 2, 2, 3])); // [1, 2, 3]

// 5. Array.of() - creates array from arguments
const arrayOfNumbers = Array.of(7); // [7] (not array with 7 empty slots!)

// 6. Using spread operator with existing array
const original = [1, 2, 3];
const copy = [...original]; // [1, 2, 3] (shallow copy)
const extended = [...original, 4, 5]; // [1, 2, 3, 4, 5]

// 7. Creating array with fill()
const filledArray = new Array(3).fill(0); // [0, 0, 0]

// 8. Array with computed values
const squares = Array(5).fill().map((_, i) => (i + 1) ** 2); // [1, 4, 9, 16, 25]

Best Practices

  • Use object literals {} for objects
  • Use array literals [] for arrays
  • Avoid new Array() with single number argument (creates empty slots)
  • Use Array.from() or Array.of() when appropriate
  • Use spread operator ... for copying and merging

Accessing & Modifying Elements

Learn how to read, update, add, and remove elements from objects and arrays.

Access Methods Comparison

Object Access
const person = { name: "John", age: 30 }; // Dot notation person.name; // "John" // Bracket notation person["age"]; // 30 // Dynamic key access const key = "name"; person[key]; // "John"
Array Access
const fruits = [ "apple", "banana", "orange" ]; // Index access (0-based) fruits[0]; // "apple" fruits[1]; // "banana" // Length property fruits.length; // 3 // Last element fruits[fruits.length - 1]; // "orange"

Modifying Objects

const user = {
  name: "Alice",
  age: 25
};

// Adding properties
user.email = "alice@example.com";  // Dot notation
user["isAdmin"] = false;          // Bracket notation

// Updating properties
user.age = 26;                     // Changed from 25 to 26

// Deleting properties
delete user.email;                // Removes email property

// Checking if property exists
"name" in user;                   // true
user.hasOwnProperty("age");       // true
user.hasOwnProperty("toString");  // false (inherited)

// Getting all property names
const keys = Object.keys(user);      // ["name", "age", "isAdmin"]
const values = Object.values(user);  // ["Alice", 26, false]
const entries = Object.entries(user); // [["name", "Alice"], ["age", 26], ...]

// Object destructuring (ES6+)
const { name, age } = user;
console.log(name);  // "Alice"
console.log(age);   // 26

Modifying Arrays

let fruits = ["apple", "banana"];

// Adding elements
fruits.push("orange");        // End: ["apple", "banana", "orange"]
fruits.unshift("kiwi");      // Start: ["kiwi", "apple", "banana", "orange"]
fruits[fruits.length] = "grape"; // End: [... , "grape"]

// Removing elements
fruits.pop();                // End: removes "grape"
fruits.shift();              // Start: removes "kiwi"

// Modifying specific index
fruits[1] = "blueberry";      // Changes index 1 to "blueberry"

// splice() - powerful method for add/remove
fruits.splice(1, 1);           // Remove 1 element at index 1
fruits.splice(1, 0, "mango");  // Add "mango" at index 1 (remove 0)
fruits.splice(1, 1, "peach"); // Replace 1 element at index 1 with "peach"

// Finding elements
const index = fruits.indexOf("apple");  // Returns index or -1
const hasBanana = fruits.includes("banana");  // true/false

// Array destructuring (ES6+)
const [first, second, ...rest] = fruits;
console.log(first);   // First element
console.log(second);  // Second element
console.log(rest);    // Array of remaining elements

Iteration & Looping

There are multiple ways to iterate over objects and arrays in JavaScript.

Array Iteration Methods

const numbers = [1, 2, 3, 4, 5];

// 1. forEach() - execute function for each element
numbers.forEach((num, index) => {
  console.log(`Index ${index}: ${num}`);
});

// 2. map() - create new array by transforming each element
const squared = numbers.map(num => num * num);
// [1, 4, 9, 16, 25]

// 3. filter() - create new array with elements that pass test
const evens = numbers.filter(num => num % 2 === 0);
// [2, 4]

// 4. reduce() - accumulate values to single result
const sum = numbers.reduce((total, num) => total + num, 0);
// 15

// 5. find() - find first element that satisfies condition
const firstEven = numbers.find(num => num % 2 === 0);
// 2

// 6. some() - check if at least one element passes test
const hasEven = numbers.some(num => num % 2 === 0);
// true

// 7. every() - check if all elements pass test
const allPositive = numbers.every(num => num > 0);
// true

Object Iteration Methods

const person = {
  name: "John",
  age: 30,
  city: "New York",
  isAdmin: false
};

// 1. for...in loop - iterate over property names
for (let key in person) {
  console.log(`${key}: ${person[key]}`);
}

// 2. Object.keys() + forEach
Object.keys(person).forEach(key => {
  console.log(`${key}: ${person[key]}`);
});

// 3. Object.values() - iterate over values
Object.values(person).forEach(value => {
  console.log(value);
});

// 4. Object.entries() - iterate over key-value pairs
Object.entries(person).forEach(([key, value]) => {
  console.log(`${key}: ${value}`);
});

// 5. Convert object to array for array methods
const personArray = Object.entries(person)
  .map(([key, value]) => `${key}=${value}`);

// ["name=John", "age=30", "city=New York", "isAdmin=false"]

// 6. Using for...of with Object.entries()
for (let [key, value] of Object.entries(person)) {
  console.log(`${key}: ${value}`);
}

// 7. Filter object properties
const filteredPerson = Object.fromEntries(
  Object.entries(person)
    .filter(([key, value]) => typeof value !== "boolean")
);

Advanced Concepts

Learn about reference vs value, copying, and other advanced topics.

Reference vs Value

// PRIMITIVE TYPES - Copied by value
let a = 10;
let b = a;  // b gets a COPY of a's value
a = 20;
console.log(a);  // 20
console.log(b);  // 10 (unchanged!)

// OBJECTS & ARRAYS - Copied by reference
let obj1 = { value: 10 };
let obj2 = obj1;  // obj2 gets a REFERENCE to obj1
obj1.value = 20;
console.log(obj1.value);  // 20
console.log(obj2.value);  // 20 (changed too!)

// ARRAYS - Also reference types
let arr1 = [1, 2, 3];
let arr2 = arr1;
arr1.push(4);
console.log(arr2);  // [1, 2, 3, 4] (changed too!)

// Checking equality
const arr3 = [1, 2, 3];
const arr4 = [1, 2, 3];

console.log(arr3 === arr4);  // false! Different references
console.log(arr3 == arr4);   // false! Different references

// Same for objects
const obj3 = { x: 1 };
const obj4 = { x: 1 };
console.log(obj3 === obj4);  // false! Different references

Copying Objects & Arrays

// SHALLOW COPY - Copies first level only

// Arrays - Shallow copy methods
const originalArray = [1, 2, [3, 4]];

// 1. Spread operator (ES6+)
const copy1 = [...originalArray];

// 2. slice()
const copy2 = originalArray.slice();

// 3. Array.from()
const copy3 = Array.from(originalArray);

// Objects - Shallow copy methods
const originalObj = { a: 1, b: { c: 2 } };

// 1. Spread operator (ES6+)
const objCopy1 = { ...originalObj };

// 2. Object.assign()
const objCopy2 = Object.assign({}, originalObj);

// DEEP COPY - Copies all nested levels

// 1. JSON methods (limitations: no functions, dates, undefined, etc.)
const deepCopy1 = JSON.parse(JSON.stringify(originalObj));

// 2. Custom recursive function
function deepCopy(obj) {
  if (typeof obj !== "object" || obj === null) {
    return obj;
  }
  
  const copy = Array.isArray(obj) ? [] : {};
  
  for (let key in obj) {
    if (obj.hasOwnProperty(key)) {
      copy[key] = deepCopy(obj[key]);
    }
  }
  
  return copy;
}

// 3. Using structuredClone() (modern browsers)
// const deepCopy3 = structuredClone(originalObj);

Shallow vs Deep Copy Warning

Shallow copies only copy the first level. Nested objects/arrays are still shared references. Use deep copy methods when you need complete independence.

// Shallow copy problem
const original = { a: 1, nested: { b: 2 } };
const shallowCopy = { ...original };

shallowCopy.a = 99;           // OK: Only changes copy
shallowCopy.nested.b = 99; // PROBLEM: Changes BOTH original and copy!

console.log(original.nested.b);  // 99 (unexpected!)

Exercise 1 Solution

// User Management System
let users = [
  { name: "Alice", email: "alice@example.com", age: 25 },
  { name: "Bob", email: "bob@example.com", age: 30 }
];

// Add new user
function addUser(name, email, age) {
  users.push({ name, email, age });
  return `User ${name} added successfully`;
}

// Find user by email
function findUserByEmail(email) {
  return users.find(user => user.email === email);
}

// Get users over certain age
function getUsersOverAge(minAge) {
  return users.filter(user => user.age > minAge);
}

// Update user email
function updateUserEmail(oldEmail, newEmail) {
  const user = findUserByEmail(oldEmail);
  if (user) {
    user.email = newEmail;
    return `Email updated for ${user.name}`;
  }
  return "User not found";
}

// Delete user by email
function deleteUserByEmail(email) {
  const initialLength = users.length;
  users = users.filter(user => user.email !== email);
  
  if (users.length < initialLength) {
    return "User deleted successfully";
  }
  return "User not found";
}

Exercise 2 Solution

// Product Catalog System
let products = [
  { id: 1, name: "Laptop", price: 999, category: "electronics" },
  { id: 2, name: "T-shirt", price: 19, category: "clothing" },
  { id: 3, name: "Smartphone", price: 699, category: "electronics" }
];

// Get products by category
function getProductsByCategory(category) {
  return products.filter(product => product.category === category);
}

// Get total inventory value
function getTotalInventoryValue() {
  return products
    .reduce((total, product) => total + product.price, 0);
}

// Find most expensive product
function getMostExpensiveProduct() {
  return products
    .reduce((max, product) => 
      product.price > max.price ? product : max, 
      products[0]
    );
}

// Apply discount to category
function applyCategoryDiscount(category, discountPercent) {
  const discountedProducts = products.map(product => {
    if (product.category === category) {
      return {
        ...product,
        price: product.price * (1 - discountPercent / 100)
      };
    }
    return product;
  });
  
  return discountedProducts;
}
