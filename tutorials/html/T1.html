Getting Started with HTML

HTML (HyperText Markup Language) is the foundational language of the web. It provides the structure and content for web pages, which is then styled with CSS and made interactive with JavaScript.

Key Insight: HTML is not a programming language but a markup language that tells browsers how to display content. Every website you visit is built with HTML at its core.

HTML Document Structure

Every valid HTML document follows a specific structure. Understanding this structure is the first step to creating web pages.

<!DOCTYPE html>
<!-- Defines the document type and HTML version -->
<html lang="en">
<head>
  <!-- Meta information about the document -->
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Page Title</title>
</head>
<body>
  <!-- All visible content goes here -->
  <h1>My First Heading</h1>
  <p>My first paragraph.</p>
</body>
</html>

Essential Elements: The <!DOCTYPE html> declaration must be the first line. The <html> element is the root container. The <head> contains metadata, and the <body> holds the visible content.

HTML Syntax Rules

HTML follows specific syntax rules that browsers use to interpret and render content correctly.

1. Elements and Tags

Most HTML elements have an opening tag, content, and a closing tag:

<tagname>Content goes here</tagname>
2. Self-Closing Elements

Some elements don't have closing tags because they don't contain content:

<!-- Line break -->
<br>

<!-- Image -->
<img src="image.jpg" alt="Description">

<!-- Input field -->
<input type="text" name="username">
3. Attributes

Attributes provide additional information about elements and are placed in the opening tag:

<a href="https://developershub.com" target="_blank" class="btn">
  Visit Developers Hub
</a>
4. Nesting Elements

HTML elements can be placed inside other elements (nested):

<div class="container">
  <h2>Section Title</h2>
  <p>
    This paragraph contains <strong>bold text</strong> and a <a href="#">link</a>.
  </p>
</div>

Complete HTML Page Example

Putting it all together, here's a complete example of a simple HTML page with semantic elements and common patterns:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Developer Portfolio</title>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <header>
    <h1>John Developer</h1>
    <p>Frontend Web Developer & UI Designer</p>
  </header>

  <nav>
    <a href="#projects">Projects</a>
    <a href="#about">About</a>
    <a href="#contact">Contact</a>
  </nav>

  <main>
    <section id="projects">
      <h2>My Projects</h2>
      <article class="project">
        <h3>E-commerce Website</h3>
        <p>Built with HTML, CSS, and JavaScript.</p>
      </article>
    </section>
  </main>

  <footer>
    <p>© 2023 John Developer. All rights reserved.</p>
  </footer>
</body>
</html>
